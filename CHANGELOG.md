# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.7] - 2026-03-20

### Added
- 🚀 Enhanced multi-chain performance for Ethereum and TRON.
- 🔐 Improved AES-256 encryption implementation for local storage.
- 📢 Added Telegram integration for real-time updates and support.
- ⚡ Optimized batch transfer engine for lower latency.
- 📊 Added new asset visualization charts in the dashboard.

### Fixed
- Fixed minor UI alignment issues on macOS.
- Improved error handling for network-related API calls.

## [1.0.0] - 2026-01-15

### Added
- 🚀 Initial release of UV Vault.
- 🔐 Support for AES-256 local encrypted storage.
- 🌐 Multi-chain support: Ethereum, BSC, TRON, and Polygon.
- ⚡ Batch transfer functionality for USDT (TRC20 & BEP20).
- 📊 Real-time balance tracking for multiple addresses.
- 🛡️ Sandboxed execution environment for secure signing.
