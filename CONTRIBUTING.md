# Contributing to UV Vault

First of all, thank you for considering contributing to UV Vault! We welcome contributions from everyone.

## How Can I Contribute?

### Reporting Bugs
- Use the [GitHub Issues](https://github.com/Upropay/UpropayVault-Wallet/issues) to report bugs.
- Provide a clear and descriptive title.
- Describe the steps to reproduce the issue.
- Mention your operating system and UV Vault version.

### Suggesting Enhancements
- Open a [GitHub Issue](https://github.com/Upropay/UpropayVault-Wallet/issues) and tag it as `enhancement`.
- Describe the feature you'd like to see and why it would be useful.

### Pull Requests
- Fork the repository and create your branch from `main`.
- Ensure your code follows the existing style and conventions.
- Provide a clear description of the changes in your PR.
- Add tests for any new functionality if applicable.
- Make sure all CI checks pass.

## Style Guide
- Follow standard formatting for the relevant programming language.
- Use meaningful variable and function names.
- Keep comments concise and informative.

## Code of Conduct
By contributing to this project, you agree to abide by our Code of Conduct. Please be respectful and professional in all interactions.
