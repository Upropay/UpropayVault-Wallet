# 使用指南 (Usage Guide)

## 1. 创建钱包 (Create Wallet)
1. 打开 UV Vault。
2. 选择“创建新钱包”。
3. 设置一个强密码，用于本地数据库加密。
4. **重要**: 记下并备份您的助记词 (Mnemonic Phrase)。**不要将助记词存储在联网的任何地方。**

## 2. 导入钱包 (Import Wallet)
1. 选择“导入现有钱包”。
2. 输入您的助记词或私钥。
3. 设置本地访问密码。

## 3. 多链管理 (Multi-Chain Management)
- UV Vault 原生支持 Ethereum, BSC, TRON 和 Polygon。
- 您可以在同一界面内查看和管理所有主网的资产。
- 点击“添加地址”可生成新的收款地址。

## 4. 批量转账 (Batch Transfer)
1. 进入“批量转账”模块。
2. 选择源钱包和目标代币 (如 USDT)。
3. 上传包含目标地址和金额的列表 (CSV 格式) 或手动输入。
4. 确认交易详情，输入密码进行本地签名。
5. 系统将自动高并发分发资产。

## 5. 本地沙盒 (Sandboxed Execution)
所有签名操作均在隔离的沙盒环境中运行，确保私钥绝不接触外界或云端。
