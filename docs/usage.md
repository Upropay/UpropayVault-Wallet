# 📖 Usage Guide (使用手册)

## 1. Create Wallet (创建钱包)
1. Open UV Vault.  
   (打开 UV Vault。)
2. Select "Create New Wallet".  
   (选择“创建新钱包”。)
3. Set a strong password for local database encryption.  
   (设置一个强密码，用于本地数据库加密。)
4. **IMPORTANT**: Write down and back up your mnemonic phrase. **Never store your mnemonic phrase anywhere online.**  
   (**重要**: 记下并备份您的助记词。不要将助记词存储在联网的任何地方。)

## 2. Import Wallet (导入钱包)
1. Select "Import Existing Wallet".  
   (选择“导入现有钱包”。)
2. Enter your mnemonic phrase or private key.  
   (输入您的助记词或私钥。)
3. Set a local access password.  
   (设置本地访问密码。)

## 3. Multi-Chain Management (多链管理)
- UV Vault natively supports Ethereum, BSC, TRON, and Polygon.  
  (UV Vault 原生支持 Ethereum, BSC, TRON 和 Polygon。)
- View and manage assets from all mainnets within the same interface.  
  (您可以在同一界面内查看和管理所有主网的资产。)
- Click "Add Address" to generate new receiving addresses.  
  (点击“添加地址”可生成新的收款地址。)

## 4. Batch Transfer (批量转账)
1. Enter the "Batch Transfer" module.  
   (进入“批量转账”模块。)
2. Select the source wallet and target token (e.g., USDT).  
   (选择源钱包和目标代币，如 USDT。)
3. Upload a list (CSV format) containing target addresses and amounts, or enter manually.  
   (上传包含目标地址和金额的列表，或手动输入。)
4. Confirm transaction details and enter your password for local signing.  
   (确认交易详情，输入密码进行本地签名。)
5. The system will automatically distribute assets with high concurrency.  
   (系统将自动高并发分发资产。)

## 5. Sandboxed Execution (本地沙盒)
All signing operations run in an isolated sandbox environment, ensuring that private keys never contact the outside world or the cloud.  
(所有签名操作均在隔离的沙盒环境中运行，确保私钥绝不接触外界或云端。)
