# ❓ FAQ (常见问题)

## 1. Is UV Vault open source? (UV Vault 是开源的吗？)
Yes, the core code of UV Vault is open-sourced under the MIT license.  
(是的，UV Vault 核心代码采用 MIT 协议开源。)

## 2. If I lose my computer, can I recover my assets? (如果我的电脑丢失了，我的资产还能找回吗？)
As long as you have saved your mnemonic phrase, your assets are safe. You can recover your assets by importing your mnemonic phrase into any BIP-39 compatible wallet software (including other copies of UV Vault, MetaMask, etc.).  
(只要您保存了助记词，您的资产就安全。您可以在任何兼容 BIP-39 标准的钱包软件中导入您的助记词来找回资产。)

## 3. Where is my private key stored? (我的私钥存储在哪里？)
Your private key is encrypted with AES-256 and stored only in the encrypted database on your local computer. It is never uploaded to our servers or any cloud service.  
(您的私钥通过 AES-256 加密后，仅存储在您本地计算机的加密数据库中。它绝不会上传到我们的服务器或任何云端。)

## 4. Why does UV Vault require a password? (为什么 UV Vault 需要密码？)
This password is used to encrypt and unlock your local wallet database. If you forget this password and have no backup of your mnemonic phrase, you will permanently lose access to the assets in that wallet.  
(该密码用于加密和解锁您的本地钱包数据库。如果您忘记了此密码，且没有备份助记词，您将永久失去对该钱包资产的访问权限。)

## 5. Which tokens does UV Vault support? (UV Vault 支持哪些代币？)
UV Vault primarily supports native assets on Ethereum, BSC, TRON, Polygon, and ERC-20, BEP-20, TRC-20 tokens (such as USDT) on those networks.  
(UV Vault 主要支持 Ethereum, BSC, TRON, Polygon 等主网的原生资产及其上的 ERC-20, BEP-20, TRC-20 代币。)

## 6. How to report bugs or get support? (如何报告 Bug 或获取支持？)
Please submit bug reports in [GitHub Issues](https://github.com/Upropay/UpropayVault-Wallet/issues), or join our Telegram channel [@upropay_wallet](https://t.me/upropay_wallet) for real-time technical support.  
(请在 GitHub Issues 中提交 Bug 报告，或加入我们的 Telegram 频道获取即时技术支持。)
