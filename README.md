upropay Vault (UV Vault) 是一款专为专业投资者和支付开发者打造的高级多链安全钱包。通过严格的“零知识”架构，确保您的私钥永不触网，真正实现“私钥即主权”。

---

## ✨ 核心特性 (Core Features)

- 🔐 **AES-256 本地加密存储**: 您的所有私钥、助记词均经过 AES-256 工业级加密后存储在本地数据库中。
- 🛡️ **隔离运行环境**: 采用自研沙盒技术，确保敏感操作在受限环境中运行，有效阻断恶意软件的窃听。
- 🌐 **Multi-Chain 原生支持**: 完美支持 Ethereum, BSC, TRON, Polygon 等主流公链。
- ⚡ **高并发批量转账**: 专为支付网关场景优化，毫秒级本地签名，一键分发资产到多个目标地址。
- 📊 **多链同屏管理**: 打破公链边界，在同一个界面内流畅管理不同主网的数千个地址。
- 🏷️ **智能分组与标记**: 为钱包打上个性化标签，按用途或项目进行多层级分组。
